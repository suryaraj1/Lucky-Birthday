## Project Description
> Lucky Birthday - If the sum of digits of a birthdate is divisible by a "lucky number", the birthday is called "Lucky Birthday"

The app requires the user to input their D.O.B as well as a lucky number. As per the definition if its a "lucky birthday" the user is notified via an alert UI component.
